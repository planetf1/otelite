# Implementation Plan: Token Usage Summary View (otelite-q3y)

## Overview
Add token usage aggregation and visualization for LLM/GenAI spans to provide cost visibility.

## Current State Analysis

### Existing Infrastructure
- ✅ GenAI span detection: `otelite-core/src/telemetry/genai.rs`
- ✅ Span storage: SQLite with JSON attributes
- ✅ JSON extraction: `json_extract()` already used in reader.rs
- ✅ API structure: Existing patterns in `otelite-server/src/api/`
- ✅ CLI structure: Existing commands in `otelite-cli/src/commands/`

### GenAI Attributes Available
From `genai.rs`, spans contain:
- `gen_ai.system` (e.g., "openai", "anthropic")
- `gen_ai.request.model` (e.g., "gpt-4", "claude-sonnet-4-20250514")
- `gen_ai.usage.input_tokens` (integer)
- `gen_ai.usage.output_tokens` (integer)
- `gen_ai.usage.total_tokens` (integer, optional)

**SQL Strategy:**
```sql
-- Overall summary
SELECT
    SUM(CAST(json_extract(attributes, '$.gen_ai.usage.input_tokens') AS INTEGER)) as total_input,
    SUM(CAST(json_extract(attributes, '$.gen_ai.usage.output_tokens') AS INTEGER)) as total_output,
    COUNT(*) as total_requests
FROM spans
WHERE json_extract(attributes, '$.gen_ai.system') IS NOT NULL
  AND start_time >= ? AND end_time <= ?;

-- By model
SELECT
    json_extract(attributes, '$.gen_ai.request.model') as model,
    SUM(CAST(json_extract(attributes, '$.gen_ai.usage.input_tokens') AS INTEGER)) as input_tokens,
    SUM(CAST(json_extract(attributes, '$.gen_ai.usage.output_tokens') AS INTEGER)) as output_tokens,
    COUNT(*) as requests
FROM spans
WHERE json_extract(attributes, '$.gen_ai.system') IS NOT NULL
  AND start_time >= ? AND end_time <= ?
GROUP BY model
ORDER BY input_tokens + output_tokens DESC;

-- By system
SELECT
    json_extract(attributes, '$.gen_ai.system') as system,
    SUM(CAST(json_extract(attributes, '$.gen_ai.usage.input_tokens') AS INTEGER)) as input_tokens,
    SUM(CAST(json_extract(attributes, '$.gen_ai.usage.output_tokens') AS INTEGER)) as output_tokens,
    COUNT(*) as requests
FROM spans
WHERE json_extract(attributes, '$.gen_ai.system') IS NOT NULL
  AND start_time >= ? AND end_time <= ?
GROUP BY system
ORDER BY input_tokens + output_tokens DESC;
```

**Implementation Notes:**
- Handle NULL values (spans without token data)
- Use CAST to convert JSON strings to integers
- Filter by time range if provided
- Return empty results if no GenAI spans found
